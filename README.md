# 3D Racing Game (Java + jMonkeyEngine)

Мини-игра в духе **Need for Speed Underground** на Java:
- загружается модель машины `bmw_racing_car.glb`;
- загружается трек `race_track.glb`;
- выводится спидометр из `speedometer-car-tachometer.png`.

## Управление
- `W` / `↑` — газ
- `S` / `↓` — тормоз / задний ход
- `A` / `←` — поворот влево
- `D` / `→` — поворот вправо

## Запуск
```bash
mvn compile
mvn exec:java
```

## Что уже реализовано
- 3D-сцена с освещением
- машинка с аркадной физикой (ускорение, торможение, повороты)
- камера преследования
- HUD-спидометр + цифровая скорость

## Ограничения
Это не буквальная копия Need for Speed Underground (что потребовало бы большой команды, лицензированных ассетов и сложной игровой логики),
а рабочий прототип с твоими 3D-ресурсами и похожим базовым геймплеем.
